# 🏆 SOBRANIE BOT - ГОТОВО!

## ✅ ЩО МИ СТВОРИЛИ:

### 📁 Файли проекту:

1. **bot.py** - головний файл, запускає бота
2. **database.py** - база даних SQLite (авто-створення таблиць)
3. **config_cog.py** - команди налаштування (`/config`)
4. **application_cog.py** - система заявок (форма + кнопки)
5. **leader_cog.py** - панель лідера для розгляду
6. **texts.py** - всі українські тексти
7. **.env** - твій токен і налаштування
8. **requirements.txt** - залежності Python
9. **vercel.json** - готово до деплою на Vercel

---

## 🚀 ЩОБ ЗАПУСТИТИ ЗАРАЗ:

### 1. Отримай Discord Token

- https://discord.com/developers/applications
- New Application → SOBRANIE Bot
- Bot → Reset Token → скопіюй

### 2. Додай бота на сервер

- OAuth2 → URL Generator
- Scopes: `bot`, `applications.commands`
- Permissions: `Administrator`
- Копіюй посилання → додай на сервер

### 3. Встав токен у .env

Відкрий `.env` і заміни:
```env
DISCORD_TOKEN=твій_токен_бота
GUILD_ID=id_твого_сервера
```

### 4. Запусти

```bash
python bot.py
```

## 🎯 ПОВНИЙ ФУНКЦІОНАЛ:

### ✅ Для користувачів:

- `/apply form` - подати заявку (модальна форма)
- `/apply status` - перевірити статус
- Постійне повідомлення з кнопкою у каналі
- Захист від повторної заявки

### ✅ Для лідерів:

- `/manage panel` - панель управління
- Фільтри заявок (всі/на розгляді/схвалені/відхилені)
- Кнопки: ✅ Схвалити | ❌ Відхилити | ⏳ На розгляді
- Перегляд деталей заявки
- Авто-видача ролей

### ✅ Налаштування:

- `/config channels` - канали
- `/config roles` - ролі
- `/config questions` - питання у формі
- `/config show` - поточні налаштування

### ✅ Автоматизація:

- Логування всіх дій
- Авто-ролі при схваленні
- Welcome повідомлення
- Перевірка активних заявок
