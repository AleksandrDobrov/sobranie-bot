# 🚀 ПОШАГОВАЯ ИНСТРУКЦИЯ ДЕПЛОЯ НА PYTHONANYWHERE

## ШАГ 1: РЕГИСТРАЦИЯ НА PYTHONANYWHERE (5 минут)

1. Откройте [PythonAnywhere.com](https://www.pythonanywhere.com/)
2. Нажмите **"Create a Free account"**
3. Заполните форму:
   - Username: придумайте имя (например: `sobraniebot`)
   - Email: ваш email
   - Password: надежный пароль
4. Подтвердите email (придет письмо)
5. Войдите в аккаунт

---

## ШАГ 2: СОЗДАНИЕ ВЕБ-ПРИЛОЖЕНИЯ (3 минуты)

1. В меню слева выберите **"Web"**
2. Нажмите **"Add a new web app"**
3. Выберите:
   - Framework: **Flask**
   - Python version: **Python 3.10**
   - Project path: оставьте по умолчанию (`/home/ваше_имя/`)
4. Нажмите **Next** → **Next** → **Create**

---

## ШАГ 3: ЗАГРУЗКА ФАЙЛОВ БОТА (10 минут)

### Способ А: Через Git (РЕКОМЕНДУЕТСЯ)

1. В консоли PythonAnywhere (меню **"Consoles"** → **"Bash"**):
```bash
# Переходим в домашнюю директорию
cd /home/ваше_имя/

# Клонируем репозиторий (замените на ваш)
git clone https://github.com/yourusername/sobranie-bot.git

# Переходим в папку бота
cd sobranie-bot

# Проверяем файлы
ls -la
```

### Способ Б: Через файловый менеджер

1. В меню выберите **"Files"**
2. Создайте папку `sobranie-bot`
3. Загрузите эти файлы с вашего компьютера:
   - `pythonanywhere_app.py`
   - `bot.py`
   - `database.py`
   - `application_cog.py`
   - `config_cog.py`
   - `leader_cog.py`
   - `texts.py`
   - `requirements_pythonanywhere.txt`
   - `.env.example`

---

## ШАГ 4: НАСТРОЙКА ОКРУЖЕНИЯ (5 минут)

1. Откройте консоль **Bash** (если еще не открыта)
2. Выполните команды:

```bash
# Переходим в папку проекта
cd /home/ваше_имя/sobranie-bot

# Создаем виртуальное окружение
python3.10 -m venv venv

# Активируем его
source venv/bin/activate

# Устанавливаем зависимости
pip install -r requirements_pythonanywhere.txt

# Создаем .env файл
nano .env
```

3. В редакторе `nano` введите:
```
DISCORD_TOKEN=ВАШ_ТОКЕН_БОТА
DISCORD_CLIENT_ID=ВАШ_CLIENT_ID
GUILD_ID=ID_ВАШЕГО_СЕРВЕРА
DATABASE_URL=./sobranie_bot.db
```

4. Нажмите `Ctrl+X`, затем `Y`, затем `Enter` для сохранения

---

## ШАГ 5: НАСТРОЙКА WSGI (3 минуты)

1. Вернитесь в меню **"Web"**
2. Найдите раздел **"WSGI configuration file"**
3. Нажмите на ссылку файла (например: `/home/ваше_имя/sobranie_bot_wsgi.py`)
4. Удалите всё содержимое и вставьте это:

```python
import sys
import os

# Добавляем путь к проекту
project_home = '/home/ваше_имя/sobranie-bot'
if project_home not in sys.path:
    sys.path.insert(0, project_home)

# Переключаемся на директорию проекта
os.chdir(project_home)

# Импортируем Flask app
from pythonanywhere_app import app as application
```

5. Нажмите **"Save"**

## 🚨 ЧТО ДЕЛАТЬ ЕСЛИ НЕ РАБОТАЕТ

### Проблема: Бот не запускается
1. Проверьте логи: **Web** → **"Logs"** → **"Error log"**
2. Убедитесь что `.env` файл создан правильно
3. Перезапустите: `https://ваше_имя.pythonanywhere.com/restart`

### Проблема: Ошибка импорта
1. Проверьте что все файлы на месте
2. Перезапустите веб-сервер: **Web** → **"Reload"**
3. Переустановите зависимости в консоли

### Проблема: Бот отключается
1. Настройте автоматический перезапуск:
   - **Tasks** → **"Add a new scheduled task"**
   - Команда: `curl https://ваше_имя.pythonanywhere.com/restart`
   - Время: каждые 10 минут
