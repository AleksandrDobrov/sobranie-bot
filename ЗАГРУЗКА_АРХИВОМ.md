# 📦 ЗАГРУЗКА БОТА ОДНИМ АРХИВОМ (2 минуты вместо 15!)

## ВАРИАНТ 1: ЧЕРЕЗ GitHub (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Загрузка кода на GitHub
1. Если еще нет - создай репозиторий на GitHub
2. Загрузи все файлы бота в репозиторий
3. В консоли PythonAnywhere:
```bash
git clone https://github.com/твой_ник/название_репозитория.git
```

---

## ВАРИАНТ 2: ЗАГРУЗКА ZIP АРХИВОМ

### Шаг 1: Создание архива на компьютере
1. Выдели все файлы бота
2. Нажми правой кнопкой → **"Отправить"** → **"Сжатую ZIP-папку"**
3. Назови архив `sobranie-bot.zip`

### Шаг 2: Загрузка архива на PythonAnywhere
1. В PythonAnywhere зайди в **"Files"**
2. Нажми **"Upload a file"**
3. Выбери твой `sobranie-bot.zip`
4. Дождись загрузки

### Шаг 3: Распаковка в консоли
1. Открой консоль **Bash**
2. Выполни команды:

```bash
# Переходим в домашнюю директорию
cd /home/твое_имя/

# Распаковываем архив
unzip sobranie-bot.zip

# Переименовываем папку если нужно
mv sobranie-bot sobranie-bot

# Заходим в папку и проверяем
cd sobranie-bot
ls -la

# Удаляем архив (не нужен больше)
rm ../sobranie-bot.zip
```

### Шаг 4: Продолжай по инструкции
Теперь продолжай с **ШАГА 4** из основной инструкции:
```bash
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements_pythonanywhere.txt
nano .env
```

---

## ВАРИАНТ 3: ЧЕРЕЗ WGET (ЕСЛИ АРХИВ НА GOOGLE DRIVE/DROPBOX)

### Шаг 1: Загрузи архив на Google Drive
1. Загрузи `sobranie-bot.zip` на Google Drive
2. Сделай файл доступным по ссылке
3. Получи прямую ссылку для скачивания

### Шаг 2: Скачивание в консоли PythonAnywhere
```bash
# Скачиваем архив
wget "прямая_ссылка_на_архив" -O sobranie-bot.zip

# Распаковываем
unzip sobranie-bot.zip

# Удаляем архив
rm sobranie-bot.zip
```

## 💡 СОВЕТЫ

- **GitHub способ** - самый надежный
- **ZIP способ** - самый быстрый для разовой загрузки
- После распаковки всегда проверяй `ls -la` что все файлы на месте
- Если папка названа иначе - замени `sobranie-bot` на правильное имя

## ⚠️ ЧТО ДЕЛАТЬ ЕСЛИ ОШИБКА

### Ошибка: "unzip: command not found"
```bash
# Устанавливаем unzip
sudo apt-get update
sudo apt-get install unzip
```

### Ошибка: "Permission denied"
```bash
# Даем права на выполнение
chmod +x *.py
```

### Ошибка: "File not found"
```bash
# Проверяем где мы находимся
pwd
ls -la
```

Выбирай любой способ - все работают! GitHub способ самый удобный для будущих обновлений.
